# シフト提出アプリ

このWebアプリは、アルバイトスタッフのシフト希望提出と、店長によるシフト作成をまとめて行うためのアプリです。

スタッフは、働ける日付と時間帯を画面上で選択して提出できます。店長は提出された希望を確認しながら、スタッフごとのシフトを作成し、完成したシフトを公開できます。

## 主な利用者

- アルバイトスタッフ
- 店長
- 管理者

## できること

- スタッフが月ごとにシフト希望を提出できる
- 日付と時間帯を選んで、入れる時間をわかりやすく登録できる
- 店長が提出状況を確認できる
- 店長がスタッフごとのシフトを作成できる
- 公開されたシフトをスタッフが確認できる
- 管理者がユーザーや提出データを確認できる
- 月ごとの運用に合わせてデータを管理できる

## 使い方の流れ

1. スタッフがログインする
2. 対象月を選ぶ
3. 入れる日付と時間帯を選んで提出する
4. 店長が提出内容を確認する
5. 店長がシフトを作成する
6. シフトを公開する
7. スタッフが公開後のシフトを確認する

## このアプリの目的

紙やメッセージで集めていたシフト希望を、画面上でまとめて管理できるようにすることを目的としています。

提出内容、割当状況、公開状態を分けて扱うことで、スタッフと店長の双方が現在の状況を確認しやすくなります。
